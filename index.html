Dragon's Dogma (Dark Arisen) - Armor Set Calculator

  1. Select vocation, or click the 'all vocations' button.
  2. Select importance weightings. (blue sliders)
  3. If required, select minimums/maximums.* (green/red sliders)
  4. Click 'Calculate' button.
*: There may be no possible sets that meet minimum/maximum criteria.

Notes

  • The number of permutations calculated is limited by pre-filtering each armor type to the ten with highest weighted score for the search. This is necessary to make searches complete in a reasonable time, but the removal of lower-score permuations may make min/max criteria impossible to meet.
  • Project is open-sourced under MIT license, you can do what you like with it.
